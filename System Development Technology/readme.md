## Reflection
 
This course gave me my first real taste of full-stack web development with a live client and a production deployment. Seeing the KADA system go from a paper form to a hosted website at `koperasikada.site` made the effort feel meaningful. Every piece of validation I wrote, every status update a member can now check from their phone instead of travelling to headquarters — that represents a real improvement to real people's lives.
 
If I could improve one thing, I would start with a shared database schema agreed on by all four members before anyone writes a single line of code. Much of the integration friction we encountered could have been avoided with that foundation in place from Sprint 1.
