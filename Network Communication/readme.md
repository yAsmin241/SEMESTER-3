
# 🌟 Overall Lab Reflection

This series of laboratory sessions provides a cohesive, hands-on mastery of network communications across the protocol stack, transitioning from capturing and analyzing raw application-layer HTTP and DNS packets using **Wireshark** 🦈 to simulating multiplexing, application port mappings, and the reliability differences between TCP and UDP in **Cisco Packet Tracer** 💻, and finally calculating and implementing structured network-layer routing via **VLSM subnetting** 🗺️. By grounding theoretical mechanics—such as the TCP three-way handshake, conditional caching behaviors, and connectionless operations—into observable simulations and direct device configurations, the coursework successfully bridges the gap between abstract architectures and practical, real-world network deployment 🚀. Ultimately, these interconnected experiments demonstrate how application-layer requests, transport delivery rules, and calculated network-layer boundaries integrate seamlessly to ensure efficient, optimized, and reliable data transmission across a shared physical infrastructure 🌐.
